📊 Telecom Churn Prediction and Risk Analysis

This is a Streamlit-based Machine Learning web application developed for predicting telecom customer churn and analysing customer risk levels. The application uses a trained Random Forest model to predict whether a customer is likely to churn based on customer service and billing details.



📂 Dataset-

The telecom customer churn dataset was used for training and testing the machine learning model. The dataset contains customer service details, billing information, and contract details used to predict customer churn behaviour.

Dataset Includes-
1. Customer tenure
2. Phone service information
3. Contract type
4. Paperless billing
5. Payment method
6. Monthly charges
7. Customer churn information


Dataset Details
1. Problem Type: Classification
2. Model Used: Random Forest Classifier
3. Prediction Type: Churn / No Churn
4. Risk Categories: Low Risk, Medium Risk, High Risk

1. Login and Authentication Page

The application includes a login and signup system with separate Admin and User access for secure dashboard usage.

2. Customer Input Dashboard

The dashboard allows users to enter telecom customer details such as tenure, contract type, payment method, and monthly charges for churn prediction.

3. Churn Prediction and Risk Analysis

The application displays churn prediction result, churn probability, risk level classification, and customer retention suggestions based on customer behaviour.

4. Churn Probability Gauge

An interactive gauge chart is used to visually represent customer churn probability and risk level.

5. Analytics Dashboard

The dashboard provides customer prediction analytics using charts, tables, and risk distribution visualization.

6. Bulk Customer Prediction

Admin users can upload CSV files to perform churn prediction for multiple customers at once and download prediction reports.

▶ Usage
1. Open the application in your browser.
2. Login as Admin or User.
3. Enter customer details in the sidebar.
4. Click on the “Predict Churn” button.
5. View churn prediction, risk level, and analytics dashboard.
6. Admin users can upload CSV files for bulk prediction.
